また、画像URLの投稿、検索が可能である。

# Features
 
フルスタックエンジニアになるためのアプリ制作第2弾。

- 画像をグリッド表示
- 画像URLを投稿
- 画像を削除
- 画像の検索

# Usage
APIサーバの立ち上げ
 
```bash
git clone git@github.com:Ryo-ta8976/grid_image_backend.git
cd grid_image_backend
docker-compose build
docker-compose up
```
